Docker Executable Image for AWS CLI
===================================

### how to install
Make sure Docker and [jenv](http://jenv.io/) installed first.

```
$ jenv repo update
$ jenv install awscli
$ docker images |grep edi/awscli
```

### how to use 

just execute aws in your terminal console.
